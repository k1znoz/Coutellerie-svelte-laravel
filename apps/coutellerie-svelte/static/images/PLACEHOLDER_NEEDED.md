# Image Placeholder Requise

Veuillez ajouter une image **placeholder.webp** (400x300px minimum) dans ce dossier.

Cette image sera affichée lorsqu'une image de couteau n'est pas disponible.

Vous pouvez :
1. Utiliser un générateur d'images en ligne (ex: Canva, Figma)
2. Convertir le fichier `placeholder.svg` existant en `.webp`
3. Utiliser une image générique de votre choix

**Dimensions recommandées** : 400x300px ou 800x600px
**Format** : WebP
**Nom du fichier** : `placeholder.webp`
